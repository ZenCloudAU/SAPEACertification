# Exam Overview

## The SAP Enterprise Architect Certification

### Structure
| Element | Detail |
|---------|--------|
| Total Questions | 40 |
| Section 1 | 22 Multiple Choice Questions |
| Section 2 | 18 questions based on Wanderlust Case Study |
| Passing Score | **80% — 32/40 correct minimum** |
| Duration | 3 hours |
| Difficulty (approx) | 25 easy / 10 moderate / 5 very hard |

---

## Domain Weightings

| # | Domain | Weight |
|---|--------|--------|
| 1 | SAP EA Framework and Toolset | 20–30% |
| 2 | Architecture Vision and Roadmap | 20–30% |
| 3 | Business Architecture | 20–30% |
| 4 | Data, Application, and Technology Architecture | 20–30% |

> All four domains are **equally weighted**. Maximum 8 wrong answers total.

---

## Exam Day Strategy

1. **Read Wanderlust case study before exam day** — it's published in advance, 18/40 questions come from it
2. **Answer all questions** — no penalty for wrong answers, never leave blanks
3. **Flag uncertain questions** — revisit at end, time is usually sufficient
4. **Sequence matters** — BMC sequence, ADM phase order, Metro Map flow are all tested
5. **Best answer, not perfect answer** — SAP exam questions look for the most appropriate response in context

## Pre-Exam Checklist

- [ ] TOGAF ADM phases memorised (Preliminary → A → B → C → D → E)
- [ ] 5 SAP EA Framework building blocks memorised
- [ ] Business Model Canvas 9 blocks in sequence memorised
- [ ] Stakeholder Map 4 quadrants + actions memorised
- [ ] Metro Map 3 variants understood
- [ ] All artifact names and their phases known
- [ ] Wanderlust case study read and annotated
- [ ] TIME model (Tolerate/Invest/Migrate/Eliminate) understood
- [ ] Cloud models (SaaS/PaaS/IaaS) + deployment types understood
- [ ] SAP Clean Core principles reviewed
- [ ] ISA-M (Integration Solution Advisory Methodology) reviewed
- [ ] Extension types (Key User / Developer / Side-by-Side) understood
